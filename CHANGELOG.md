# Changelog

All notable changes to the Ellison Electronics project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-01

### Added
- Initial release of Ellison Electronics e-commerce platform
- User registration and authentication system
- Product catalog with search and filtering
- Shopping cart functionality
- Order processing and management
- Admin dashboard for product management
- Email notifications for orders and registrations
- Comprehensive documentation (README, ARCHITECTURE, CONTRIBUTING)
- Professional project structure and configuration files
- `.gitignore`, `.gitattributes`, and `LICENSE` files
- `.env.example` for environment configuration
- Demo assets directory structure
- Javadoc comments for key utility and service classes

### Features
- **User Management**
  - Customer registration with email validation
  - Secure login/logout
  - User profile management
  - Session-based authentication

- **Product Management**
  - Add, update, delete products (Admin)
  - Product images and descriptions
  - Search products by keyword
  - Filter products by category/type
  - Stock management

- **Shopping Cart**
  - Add/remove items from cart
  - Update item quantities
  - Real-time cart total calculation
  - Persistent cart storage

- **Order Processing**
  - Create orders from cart
  - Order tracking
  - Shipment management
  - Transaction records

- **Email Notifications**
  - Registration confirmation emails
  - Order confirmation emails
  - Shipment status updates

### Technical Stack
- Java 8
- JSP (JavaServer Pages)
- Servlets
- MySQL 8.0
- Apache Tomcat 9+
- Bootstrap 3
- jQuery
- Maven

- `CONTRIBUTING.md` - Contribution guidelines
