## How to Add Screenshots

1. Take a screenshot of the application
2. Save as PNG format (recommended: 1280x720 or higher)
3. Place in this directory
4. Update the README.md with the screenshot reference

## Demo GIFs

For animated demonstrations, create GIFs showing:
- User registration flow
- Product search and filtering
- Adding items to cart
- Checkout process
- Admin product management

Tools for creating GIFs:
- **ScreenToGif** (Windows)
- **Gifox** (macOS)
- **Peek** (Linux)
- **FFmpeg** (Command line)

Example FFmpeg command:
```bash
ffmpeg -i video.mp4 -vf "fps=10,scale=1280:-1:flags=lanczos" demo.gif
```
